# WFnets
This repository contains the implementation of an algorithm for obtaining the trail-based history equivalence mapping for WF nets.

arXiv: https://arxiv.org/abs/2412.08314

## Files

- `HistEqMapping.cpp`: The main program to obtain the mapping
- Folder `example_inputs`: Some example inputs to the program in the required format
- The pre-print of this work as submitted on arXiv

## To run
```
g++ HistEqMapping.cpp
./a.out <oldWFnetFile> <newWFnetFile>
```
